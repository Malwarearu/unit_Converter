# Unit Converter

## Overview

Welcome to the **Unit Converter** program! This simple C application allows users to convert units in three categories: Temperature, Mass, and Length.

## Features

### Temperature Conversion

1. Celsius to Fahrenheit
2. Fahrenheit to Celsius
3. Celsius to Kelvin
4. Kelvin to Celsius
5. Fahrenheit to Kelvin
6. Kelvin to Fahrenheit

### Mass Conversion

1. Kilogram to Pound
2. Pound to Kilogram
3. Kilogram to Gram
4. Gram to Kilogram
5. Pound to Gram
6. Gram to Pound

### Length Conversion

1. Meter to Inch
2. Inch to Meter
3. Meter to Foot
4. Foot to Meter
